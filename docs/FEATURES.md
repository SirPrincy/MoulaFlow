# Guide des Fonctionnalités - Moula Flow

Moula Flow est conçu pour être simple, rapide et élégant. Voici un tour d'horizon de ses capacités.

## 📊 Tableau de Bord (Dashboard)
Le point d'entrée de l'application. Il est entièrement configurable.
- **Soldes Globaux** : Vue d'ensemble de tous vos comptes.
- **Flux (Flow)** : Comparaison visuelle entre vos revenus et vos dépenses sur le mois en cours.
- **Catégories** : Répartition de vos dépenses par catégorie (Donut chart).
- **Transactions Récentes** : Accès rapide aux 5 dernières opérations.
- **Configuration** : Cliquez sur l'icône de réglage pour masquer ou réorganiser ces blocs.

## 💳 Gestion des Portefeuilles (Wallets)
Moula Flow supporte trois types de portefeuilles :
- **Standard (Courant)** : Pour vos opérations quotidiennes.
- **Épargne (Savings)** : Avec possibilité de définir un **montant cible** pour suivre vos économies via une barre de progression.
- **Dette (Debt)** : Pour suivre ce que vous devez ou ce qu'on vous doit, avec gestion de la **date d'échéance** et statut de règlement.

## 🏷️ Catégories & Tags
L'organisation est au cœur de l'application.
- **Hiérarchie** : Chaque catégorie (ex: Alimentation) peut avoir des sous-catégories (ex: Supermarché, Restaurant).
- **Recherche** : Le sélecteur modal permet de trouver instantanément une catégorie par son nom.
- **Tags** : Ajoutez des étiquettes (ex: #vacances2024) séparées par des points-virgules pour des analyses croisées.

## 🔁 Transactions Récurrentes
- Marquez une transaction comme "Récurrente" lors de sa création.
- Retrouvez-les toutes dans la page dédiée accessible depuis le menu latéral pour un suivi facilité de vos charges fixes.

## 🌙 Personnalisation
- **Thème** : Bascule automatique ou manuelle entre le Mode Sombre (Batman Style/Premium) et le Mode Clair.
- **Reset des Données** : Dans les paramètres, vous pouvez effacer toutes les données pour repartir de zéro.
